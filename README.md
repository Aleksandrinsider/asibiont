# AI Task Management Telegram Bot

Этот бот позволяет общаться с ИИ на естественном языке для управления задачами, получения напоминаний и совершения платежей. Все взаимодействие происходит через диалог, без команд.

## Функции

- Добавление задач через разговор
- Перечисление задач
- Завершение задач
- Установка напоминаний
- Платежи через Yookassa
- **Веб-интерфейс для пользователей**: Панель управления задачами в стиле GitHub Issues

## Веб-интерфейс

Пользователи могут открыть веб-версию через Telegram, нажав кнопку "Открыть веб-версию" в боте. Интерфейс имитирует GitHub с темной темой и показывает задачи как Issues:

- Список задач с статусами (Open/Closed)
- Фильтры по статусу
- Приоритеты задач
- Даты создания и дедлайны

URL: `{WEB_APP_URL}/issues?telegram_id={user_telegram_id}`

Для локального тестирования: `http://localhost:8000/issues?telegram_id={user_id}`

## Установка

1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте переменные окружения в .env:
   - TELEGRAM_TOKEN: Токен бота от BotFather
   - DEEPSEEK_API_KEY: API ключ от DeepSeek
   - REDIS_URL: URL Redis базы
   - DATABASE_URL: URL PostgreSQL базы
   - YOOKASSA_SHOP_ID: ID магазина Yookassa
   - YOOKASSA_SECRET_KEY: Секретный ключ Yookassa
   - WEBHOOK_URL: URL вебхука для Railway
3. Для локального тестирования: `LOCAL=1 python main.py`
4. Для продакшена на Railway: разверните с переменными окружения

## Развертывание на Railway

1. Создайте проект на Railway
2. Подключите GitHub репозиторий
3. Установите переменные окружения
4. Railway автоматически развернет бота

## Веб-интерфейс

Пользователи могут получить доступ к персональной панели управления через команду `/dashboard` в боте. Панель показывает:

- Общую статистику задач (всего, завершено, в ожидании, пропущено)
- Среднее время завершения задач
- Статус подписки
- Последние взаимодействия с ботом

URL панели: `{WEBHOOK_URL}/dashboard?telegram_id={user_telegram_id}`

Также доступна веб-версия в стиле GitHub Issues для просмотра задач.

## Тестирование

Запустите `python test_dialogue.py` для симуляции диалога.
Запустите `python test_dashboard.py` для тестирования веб-интерфейса.
