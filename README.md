# AI Агент для Управления Задачами 🤖

AI-powered Telegram бот для интеллектуального управления задачами с обработкой естественного языка, автоматическими напоминаниями и социальными функциями.

## 📊 Статус проекта

### ✅ Готово к продакшену
- **Архитектура**: Гибридная система (правила + AI) для надежности
- **Тестирование**: Полный тест выполнения команд - 100% успех
- **Надежность**: Без галлюцинаций, все команды выполняются в БД
- **Производительность**: Быстрая маршрутизация без лишних AI вызовов
- **Документация**: Полная спецификация команд и API

### 🔧 Технические достижения
- **AI Intent Classification**: Классификация намерений с 100% точностью
- **Command-Based Architecture**: Надежная обработка команд без галлюцинаций
- **Database Integration**: Полная интеграция с PostgreSQL
- **Time Parsing**: Умный парсинг времени с поддержкой часовых поясов
- **Error Handling**: Корректная обработка всех типов данных

## 🌟 Возможности

### Основной Функционал
- **Управление задачами на естественном языке**: Создание, редактирование и отслеживание задач через разговорный ИИ
- **Умные напоминания**: Автоматическое планирование и проактивные уведомления
- **Система подписок**: Тарифы LIGHT/STANDARD/PREMIUM с разными возможностями
- **Профили пользователей**: Персонализированный опыт с отслеживанием навыков, интересов и целей
- **Социальные функции**: Поиск контактов и делегирование задач

### AI Возможности
- **Контекстно-зависимые ответы**: Адаптация поведения в зависимости от типа сообщения
- **Интеграция инструментов**: Специализированные функции для операций с задачами
- **Система памяти**: Сохранение предпочтений пользователя и истории разговоров
- **Многоязычная поддержка**: Работа с русским и английским языками

### Автоматизация
- **Сервис автопостинга**: Генерация ежедневных постов о прогрессе с помощью ИИ
- **Сервис напоминаний**: Фоновое планирование задач и уведомления
- **Миграции базы данных**: Автоматические обновления схемы

### Финансовый Анализ (PREMIUM)
- **Мониторинг активов**: Отслеживание цен металлов, валют и акций в реальном времени
- **Технический анализ**: Автоматический расчет RSI, MACD, Bollinger Bands
- **Анализ объемов**: Мониторинг объема торгов для выявления значимых движений

### Worker Задачи (PREMIUM)
- **Фоновый мониторинг**: Автоматизированные задачи выполняются минимум раз в сутки
- **Мониторинг активов**: Металлы (GOLD, SILVER), валюты (EURUSD, USDRUB), акции (AAPL, TSLA)
- **Мониторинг погоды**: Температура и погодные условия по городам
- **Стили ответов**: Формальные отчеты или естественные разговорные сообщения
- **Типы анализа**: Простой мониторинг цен, технический анализ, анализ объемов

## 🎯 Worker Задачи (PREMIUM)

Пользователи могут использовать **worker задачи** для автоматизации различных типов мониторинга. Вот все доступные варианты:

### 📈 Мониторинг финансовых активов (`monitor_asset`)
**Что мониторит:** Металлы, валюты, акции
**Доступные активы:**
- **Металлы:** `GOLD` (золото), `SILVER` (серебро)
- **Валюты:** Любые пары типа `EURUSD`, `USDRUB` 
- **Акции:** Любые тикеры типа `AAPL`, `GOOGL`, `TSLA`

**Типы анализа:**
- `price_monitoring` - простой мониторинг цены (уведомление при падении ниже порога)
- `technical_analysis` - **комплексный анализ с полным набором индикаторов**: RSI, MACD, Bollinger Bands, SMA/EMA, Stochastic, ADX, CCI, MFI, OBV + новостной сентимент + AI рекомендации
- `volume_analysis` - анализ объема торгов (для акций)

**Примеры команд:**
```
"Мониторь золото, если цена ниже 2000, уведомляй каждый день"
"Создай технический анализ AAPL с разговорным стилем"
"Следи за объемом торгов серебра"
```

### 🌤️ Мониторинг погоды (`monitor_weather`)
**Что мониторит:** Температура и погодные условия по городам
**Условия уведомлений:**
- Температура ниже/выше порога
- Конкретные погодные условия (дождь, снег, etc.)

**Примеры команд:**
```
"Мониторь погоду в Москве, если температура ниже -5"
"Уведомляй о дожде в Санкт-Петербурге"
"Следи за погодой в Сочи, если будет снег"
```

### 📊 Специализированный мониторинг золота (`monitor_gold_market`)
**Что делает:** Простой мониторинг цены золота (унаследованный от старой версии)
**Особенности:** Только для золота, без технического анализа

### 🧠 Глубокий анализ активов
При выборе `technical_analysis` система выполняет **комплексный анализ**:

#### 📈 Технический анализ
- **RSI (Relative Strength Index)**: Определение перекупленности/перепроданности
- **MACD (Moving Average Convergence Divergence)**: Анализ трендов и моментума
- **Bollinger Bands**: Оценка волатильности и уровней поддержки/сопротивления
- **Объем торгов**: Анализ активности и ликвидности

#### 📰 Новостной анализ
- **Сентимент новостей**: Автоматический анализ тональности новостей
- **Релевантность**: Фильтрация новостей по степени связи с активом
- **Влияние на рынок**: Учет новостного фона при формировании рекомендаций
- **Ключевые события**: Выделение наиболее важных новостей

#### 🤖 AI рекомендации
- **Комплексная оценка**: Синтез технических и фундаментальных факторов
- **Риск-менеджмент**: Оценка волатильности и потенциальных потерь
- **Временной горизонт**: Рекомендации с учетом кратко-/долгосрочных перспектив
- **Диверсификация**: Советы по управлению портфелем

**Специализированный AI промпт:**
Финансовый анализ использует **отдельный AI промпт**, оптимизированный для:
- Профессионального финансового анализа
- Синтеза технических и новостных данных
- Оценки рисков и рекомендаций
- Этически корректных формулировок (всегда напоминает, что это не финансовый совет)

### 🎨 Стили ответов
Все worker задачи поддерживают два стиля ответов:
- `formal` - формальные отчеты с эмодзи и структурой
- `conversational` - **естественные, разговорные сообщения, генерируемые AI** (используют тот же промпт, что и обычные задачи для consistency)

**AI Генерация ответов:**
При выборе `conversational` стиля worker задачи используют тот же AI промпт, что и обычные задачи. Это обеспечивает:
- Единообразие стиля общения бота
- Персонализированные, естественные ответы
- Контекстно-зависимые советы и рекомендации
- Дружелюбный тон без формальностей

### 🔧 Как создать worker задачу

**Через чат-бот:**
```
"Создай фоновую задачу для мониторинга золота"
"Мониторь акции Apple каждый день"
"Следи за погодой в Москве"
```

**Через API (для разработчиков):**
```python
create_worker_task(
    task_description="Мониторинг рынка",
    action="monitor_asset",  # или "monitor_weather"
    asset_type="stock",      # metal/currency/stock
    symbol="AAPL",
    analysis_type="technical_analysis",
    response_style="conversational",
    threshold=150.0,
    interval_minutes=1440
)
```

### ⚙️ Технические детали
- **Интервал:** Минимум 1 час (60 минут)
- **Доступ:** Только PREMIUM подписка
- **Лимит:** Неограниченное количество worker задач
- **Хранение:** В таблице Task с префиксом "Worker:"
- **Уведомления:** Через Telegram бота

### 📱 Управление
- **Просмотр:** Фильтр "Автоматические" в дашборде
- **Удаление:** Команда "Удали фоновую задачу"
- **Статус:** Всегда активны до удаления

Worker задачи позволяют пользователям **автоматизировать рутинный мониторинг** и получать своевременные уведомления без постоянной проверки! 🚀

### Предварительные Требования
- Python 3.10+
- PostgreSQL (продакшн) или SQLite (разработка)
- Токен Telegram Bot
- Ключ API DeepSeek

### Установка

1. **Клонировать репозиторий**
   ```bash
   git clone <repository-url>
   cd task-management-agent
   ```

2. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настроить окружение**
   ```bash
   cp .env.example .env
   ```

## 🔧 Конфигурация

### Переменные Окружения

| Переменная | Описание | Обязательно |
|------------|----------|-------------|
| `TELEGRAM_TOKEN` | Токен Telegram Bot API | Да (прод) |
| `DEEPSEEK_API_KEY` | Ключ API DeepSeek | Да |
| `DATABASE_URL` | Строка подключения PostgreSQL | Да (прод) |
| `SESSION_SECRET` | Ключ шифрования сессий | Да |
| `YOOKASSA_SHOP_ID` | ID магазина YooKassa | Да (прод) |
| `YOOKASSA_SECRET_KEY` | Секретный ключ YooKassa | Да (прод) |
| `OPENWEATHERMAP_API_KEY` | Ключ OpenWeatherMap API (для мониторинга погоды) | Нет (PREMIUM) |
| `ALPHA_VANTAGE_API_KEY` | Ключ Alpha Vantage (для мониторинга рынков и индикаторов) | Нет (PREMIUM) |
| `LOCAL` | Включить режим локальной разработки | Нет |

### Тарифы Подписок

- **LIGHT**: Базовое управление задачами
- **STANDARD**: Продвинутые функции + делегирование
- **PREMIUM**: Премиум функции + неограниченное делегирование + **фоновые задачи (workers)**

### PREMIUM Функции

#### Фоновые Задачи (Workers)
Только для пользователей с подпиской PREMIUM доступны автоматизированные фоновые задачи:

**Мониторинг погоды:**
- Отслеживание температуры и погодных условий
- Уведомления при достижении пороговых значений
- Настройка города и условий (дождь, снег, etc.)

**Мониторинг рынков:**
- Отслеживание цен на золото
- Уведомления о благоприятных возможностях для покупки

**Требуемые API ключи:**
```bash
# Для мониторинга погоды
OPENWEATHERMAP_API_KEY=ваш_ключ_от_openweathermap.org

# Для мониторинга золота  
ALPHA_VANTAGE_API_KEY=ваш_ключ_от_alphavantage.co
```

**Примеры команд:**
- "Создай worker для мониторинга погоды в Москве, уведомляй если температура ниже 0"
- "Настрой фоновую задачу: следить за золотом, сообщать если цена ниже 2000 долларов"
- "Создай технический анализ акций AAPL с RSI и MACD"
- "Мониторь объем торгов AAPL, уведомляй при высоком объеме"
- "Анализируй рынок золота с индикаторами Bollinger Bands"

### Промокоды
