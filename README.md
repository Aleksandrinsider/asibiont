### 🚀 Новая Архитектура (январь 2026)

**Гибридный Автономный Агент** - революционный подход к AI-ассистенту:

```
Запрос → [Планирование] → [Выполнение] → [Рефлексия] → Ответ
```

**Ключевые преимущества**:
- ✅ **Автономное планирование**: AI сам решает, какие действия выполнить
- ✅ **Надежное выполнение**: Использует проверенные handlers, а не генерирует код
- ✅ **Естественные ответы**: Рефлексия над результатами для человечных ответов
- ✅ **Адаптация к контексту**: Учитывает историю разговора и текущие задачи
- ✅ **Быстрая работа**: Без генерации кода - мгновенное выполнение

📖 **Подробнее**: [ai_integration/HYBRID_AGENT_ARCHITECTURE.md](ai_integration/HYBRID_AGENT_ARCHITECTURE.md)

## 🎯 Worker Задачи (PREMIUM)

Пользователи могут использовать **worker задачи** для автоматизации различных типов мониторинга. Вот все доступные варианты:

### 📈 Мониторинг финансовых активов (`monitor_asset`)
**Что мониторит:** Металлы, валюты, акции
**Доступные активы:**
- **Металлы:** `GOLD` (золото), `SILVER` (серебро), `PLAT` (платина), `PALL` (палладий)
- **Нефть:** `WTI` (нефть WTI), `BRENT` (нефть Brent)
- **Валюты:** `EURUSD`, `GBPUSD`, `USDJPY`, `USDCHF`, `AUDUSD`, `USDCAD`, `NZDUSD`, `USDRUB`

**Анализ:** Комплексный технический анализ с индикаторами и новостями

**Анализ:** Комплексный технический анализ с индикаторами и новостями

**Примеры команд:**
```
"Создай технический анализ WTI с разговорным стилем"
"Анализируй EURUSD техническими индикаторами"
"Мониторь золото с комплексным анализом"
```

### 🌤️ Мониторинг погоды (`monitor_weather`)
**Что мониторит:** Температура и погодные условия по городам
**Условия уведомлений:**
- Температура ниже/выше порога
- Конкретные погодные условия (дождь, снег, etc.)

**Примеры команд:**
```
"Мониторь погоду в Москве, если температура ниже -5"
"Уведомляй о дожде в Санкт-Петербурге"
"Следи за погодой в Сочи, если будет снег"
```

### 🧠 Глубокий анализ активов
При выборе `technical_analysis` система выполняет **комплексный анализ**:

#### 📈 Технический анализ (оптимизированный для лимита API)
- **RSI (Relative Strength Index)**: Определение перекупленности/перепроданности (рассчитывается локально)
- **MACD (Moving Average Convergence Divergence)**: Анализ трендов и моментума (упрощенная версия)
- **Bollinger Bands**: Оценка волатильности и уровней поддержки/сопротивления (рассчитывается локально)
- **SMA/EMA (Moving Averages)**: Анализ трендов на основе скользящих средних (рассчитывается локально)

#### 📰 Новостной анализ (ограниченный)
- **Сентимент новостей**: Автоматический анализ тональности (максимум 3 новости)
- **Релевантность**: Фильтрация новостей по степени связи с активом
- **Влияние на рынок**: Учет новостного фона при формировании рекомендаций

#### 🤖 AI рекомендации
- **Комплексная оценка**: Синтез технических индикаторов и новостного сентимента
- **Риск-менеджмент**: Оценка волатильности и потенциальных рисков
- **Временной горизонт**: Рекомендации с учетом кратко-/долгосрочных перспектив
- **Диверсификация**: Советы по управлению портфелем

**Специализированный AI промпт:**
Финансовый анализ использует **отдельный AI промпт**, оптимизированный для:
- Профессионального финансового анализа
- Синтеза технических и новостных данных
- Оценки рисков и рекомендаций
- Этически корректных формулировок (всегда напоминает, что это не финансовый совет)

### 🎨 Стили ответов
Все worker задачи поддерживают два стиля ответов:
- `formal` - формальные отчеты с эмодзи и структурой
- `conversational` - **естественные, разговорные сообщения, генерируемые AI** (используют тот же промпт, что и обычные задачи для consistency)

**AI Генерация ответов:**
При выборе `conversational` стиля worker задачи используют тот же AI промпт, что и обычные задачи. Это обеспечивает:
- Единообразие стиля общения бота
- Персонализированные, естественные ответы
- Контекстно-зависимые советы и рекомендации
- Дружелюбный тон без формальностей

### 🔧 Как создать worker задачу

**Через чат-бот:**
```
"Создай фоновую задачу для мониторинга золота"
"Мониторь акции Apple каждый день"
"Следи за погодой в Москве"
```

**Через API (для разработчиков):**
```python
create_worker_task(
    task_description="Мониторинг рынка",
    action="monitor_asset",  # или "monitor_weather"
    asset_type="stock",      # metal/currency/stock
    symbol="AAPL",
    analysis_type="technical_analysis",
    response_style="conversational",
    threshold=150.0,
    interval_minutes=1440
)
```

### ⚙️ Технические детали
- **Интервал:** Минимум 1 час (60 минут)
- **Доступ:** Только PREMIUM подписка
- **Лимит:** Неограниченное количество worker задач
- **Хранение:** В таблице Task с префиксом "Worker:"
- **Уведомления:** Через Telegram бота

### 📱 Управление
- **Просмотр:** Фильтр "Автоматические" в дашборде
- **Удаление:** Команда "Удали фоновую задачу"
- **Статус:** Всегда активны до удаления

Worker задачи позволяют пользователям **автоматизировать рутинный мониторинг** и получать своевременные уведомления без постоянной проверки! 🚀

### Тарифы Подписок

- **LIGHT**: Управление задачами + погода и новости по городам
- **STANDARD**: Всё из Light + делегирование + координация команд
- **PREMIUM**: Всё из Standard + автоматизация на автопилоте + фоновые задачи (workers)

### PREMIUM Функции

#### Фоновые Задачи (Workers)
Только для пользователей с подпиской PREMIUM доступны автоматизированные фоновые задачи:

**Мониторинг погоды (автоматический):**
- Отслеживание температуры и погодных условий
- Уведомления при достижении пороговых значений
- Настройка города и условий (дождь, снег, etc.)

**Мониторинг рынков:**
- Отслеживание цен на золото
- Уведомления о благоприятных возможностях для покупки

**Требуемые API ключи для ВСЕХ тарифов:**
