# AI Task Management Telegram Bot

Этот бот позволяет общаться с ИИ на естественном языке для управления задачами, получения напоминаний и совершения платежей. Все взаимодействие происходит через диалог, без команд.

## AI-классификация намерений

Система использует DeepSeek AI для интеллектуального анализа сообщений пользователя и определения их намерений. Это позволяет более точно понимать контекст и автоматически выполнять соответствующие действия.

### Поддерживаемые типы намерений:

- **add_task**: Создание новых задач ("Напомни позвонить маме завтра")
- **complete_task**: Отметка задач как выполненных ("Я сделал уборку")
- **list_tasks**: Показ списка задач ("Покажи мои задачи")
- **delegate_task**: Делегирование задач ("Поручи @user проверить отчет")
- **update_profile**: Обновление профиля ("Мой город Москва")
- **chat**: Обычное общение и вопросы

### Преимущества AI-классификации:

- **Гибкость**: Понимает разнообразные формулировки
- **Контекст**: Учитывает нюансы и неявные указания
- **Точность**: Автоматически извлекает параметры (время, названия задач)
- **Адаптивность**: Улучшается с каждым взаимодействием

### Примеры работы:

```
"Напомни сходить в магазин завтра в 15:00"
→ AI определяет: add_task, task_title="Сходить в магазин", reminder_time="завтра в 15:00"

"Я закончил проект"
→ AI определяет: complete_task, task_title="проект"

"Что у меня запланировано?"
→ AI определяет: list_tasks
```

## Функции

- **Интеллектуальное формулирование задач**: AI помогает создавать конкретные и детальные задачи вместо общих формулировок
- Добавление задач через разговор
- Перечисление задач
- Завершение задач
- Установка напоминаний
- Делегирование задач другим пользователям
- Поиск партнёров по интересам
- Платежи через Yookassa
- **Веб-интерфейс**: Дашборд в стиле GitHub для управления задачами с авторизацией через Telegram
- **Голосовое управление**: Полностью голосовое взаимодействие с AI через микрофон

## Интеллектуальное формулирование задач

AI агент помогает формулировать задачи правильно, задавая уточняющие вопросы:

### Примеры улучшения формулировок:

❌ **Плохо**: "проверить почту"  
✅ **Хорошо**: "Проверить почту на предмет ответа от клиента Иванова по договору"

❌ **Плохо**: "позвонить другу"  
✅ **Хорошо**: "Позвонить Марии обсудить договор на поставку оборудования"

❌ **Плохо**: "встретиться с командой"  
✅ **Хорошо**: "Встреча с командой: обсудить квартальные результаты и план на Q2"

### Как это работает:

```
Пользователь: "Добавь задачу позвонить другу"
AI: "О чём нужно поговорить? Это рабочий или личный звонок?"
Пользователь: "Про поездку на выходные"
AI: "✅ Добавил: Позвонить другу обсудить план поездки на выходные. На какое время?"
```

## Голосовое управление

Веб-интерфейс поддерживает голосовой ввод:

- **Голосовой ввод**: Нажмите кнопку микрофона 🎤 в поле чата для голосового сообщения
- **Автоматическая отправка**: Распознанная речь автоматически отправляется AI
- **Русский язык**: Поддержка распознавания русской речи

Требования:
- Современный браузер с поддержкой Web Speech API (Chrome, Edge, Safari)
- Доступ к микрофону
- Разрешение на использование микрофона в браузере

## Установка

1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте переменные окружения в .env:
   - TELEGRAM_TOKEN: Токен бота от BotFather
   - TELEGRAM_BOT_USERNAME: Username бота (например, @asibiont_bot)
   - DEEPSEEK_API_KEY: API ключ от DeepSeek
   - REDIS_URL: URL Redis базы
   - DATABASE_URL: URL PostgreSQL базы
   - YOOKASSA_SHOP_ID: ID магазина Yookassa
   - YOOKASSA_SECRET_KEY: Секретный ключ Yookassa
   - WEBHOOK_URL: URL вебхука для Railway
   - FREE_ACCESS_MODE: "False" для боевого режима (требуется подписка), "True" для бесплатного доступа
3. Для локального тестирования: `LOCAL=1 python main.py` (веб-сервер на http://localhost:8080)
4. Для продакшена на Railway: разверните с переменными окружения (FREE_ACCESS_MODE=False)

## Развертывание на Railway

1. Создайте проект на Railway
2. Подключите GitHub репозиторий
3. Установите переменные окружения
4. Railway автоматически развернет бота

## Веб-интерфейс

Веб-приложение доступно по URL вебхука (для Railway) или localhost:8080 (локально).

- **Главная страница (/)**: Вход через Telegram Login Widget
- **Дашборд (/dashboard)**: Список задач пользователя в стиле GitHub

Для авторизации через Telegram:
1. Зарегистрируйте домен в BotFather: `/setdomain yourdomain.com`
2. Пользователи могут войти, нажав кнопку "Войти через Telegram"

## Тестирование

Запустите `python test_dialogue.py` для симуляции диалога.
Для тестирования веб-интерфейса откройте браузер на соответствующем URL.
