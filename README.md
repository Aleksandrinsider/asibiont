# AI Task Management Telegram Bot

Этот бот позволяет общаться с ИИ, управлять задачами, получать напоминания и совершать платежи.

## Установка

1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте переменные окружения в .env
3. Запустите: `python main.py`

## Команды

- /start: Начать
- /add_task Название: Описание - Добавить задачу
- /list_tasks - Список задач
- /complete_task <id> - Выполнить задачу
- /set_reminder <id> <YYYY-MM-DD HH:MM> - Установить напоминание
- /pay - Создать платеж

Для чата с ИИ просто отправьте сообщение.
