# AI Агент для Управления Задачами 🤖

AI-powered Telegram бот для интеллектуального управления задачами с обработкой естественного языка, автоматическими напоминаниями и социальными функциями.

## 🌟 Возможности

### Основной Функционал
- **Управление задачами на естественном языке**: Создание, редактирование и отслеживание задач через разговорный ИИ
- **Умные напоминания**: Автоматическое планирование и проактивные уведомления
- **Система подписок**: Тарифы LIGHT/STANDARD/PREMIUM с разными возможностями
- **Профили пользователей**: Персонализированный опыт с отслеживанием навыков, интересов и целей
- **Социальные функции**: Поиск контактов и делегирование задач

### AI Возможности
- **Контекстно-зависимые ответы**: Адаптация поведения в зависимости от типа сообщения
- **Интеграция инструментов**: Специализированные функции для операций с задачами
- **Система памяти**: Сохранение предпочтений пользователя и истории разговоров
- **Многоязычная поддержка**: Работа с русским и английским языками

### Автоматизация
- **Сервис автопостинга**: Генерация ежедневных постов о прогрессе с помощью ИИ
- **Сервис напоминаний**: Фоновое планирование задач и уведомления
- **Миграции базы данных**: Автоматические обновления схемы

## 🚀 Быстрый Старт

### Предварительные Требования
- Python 3.10+
- PostgreSQL (продакшн) или SQLite (разработка)
- Токен Telegram Bot
- Ключ API DeepSeek

### Установка

1. **Клонировать репозиторий**
   ```bash
   git clone <repository-url>
   cd task-management-agent
   ```

2. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настроить окружение**
   ```bash
   cp .env.example .env
   # Отредактировать .env с вашими API ключами и настройками
   ```

4. **Инициализировать базу данных**
   ```bash
   python init_db.py
   ```

### Запуск Приложения

**Режим разработки:**
```bash
LOCAL=1 python main.py
```

**Продакшн режим:**
```bash
python main.py
```

**Сервис автопостинга:**
```bash
python auto_post_service.py
```

## 🔧 Конфигурация

### Переменные Окружения

| Переменная | Описание | Обязательно |
|------------|----------|-------------|
| `TELEGRAM_TOKEN` | Токен Telegram Bot API | Да (прод) |
| `DEEPSEEK_API_KEY` | Ключ API DeepSeek | Да |
| `DATABASE_URL` | Строка подключения PostgreSQL | Да (прод) |
| `SESSION_SECRET` | Ключ шифрования сессий | Да |
| `YOOKASSA_SHOP_ID` | ID магазина YooKassa | Да (прод) |
| `YOOKASSA_SECRET_KEY` | Секретный ключ YooKassa | Да (прод) |
| `LOCAL` | Включить режим локальной разработки | Нет |

### Тарифы Подписок

- **LIGHT**: Базовое управление задачами
- **STANDARD**: Продвинутые функции + делегирование
- **PREMIUM**: Премиум функции + неограниченное делегирование

## 🚢 Развертывание

2. Установить переменные окружения

3. Настроить переменные окружения


## 📡 API Endpoints

### Веб Интерфейс
- `GET /` - Страница входа
- `GET /dashboard` - Главная панель
- `POST /webhook` - Telegram webhook

### Админ Endpoints
- `GET /admin/users` - Управление пользователями
- `GET /admin/security` - Панель безопасности
- `GET /clear_database` - Очистка базы данных

### Интеграция Платежей
- `POST /yookassa/webhook` - Уведомления о платежах
- `GET /create_payment` - Создание платежа
- `GET /subscription_tiers` - Опции подписок
