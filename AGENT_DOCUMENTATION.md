# Документация AI-агента для управления задачами

## Обзор

AI-агент — это интеллектуальный помощник в Telegram-боте для управления задачами. Агент использует продвинутые возможности ИИ для естественного общения, автоматизации процессов и персонализированной помощи пользователям.

## Ключевые возможности агента

### 1. Управление задачами
- **Создание задач**: Автоматическое распознавание запросов на создание задач из естественного текста
- **Редактирование**: Изменение параметров задач (время, описание, статус)
- **Удаление**: Удаление задач с запросом причины
- **Отслеживание статуса**: АКТУАЛЬНО/ПРОСРОЧЕНО/ВЫПОЛНЕНО/ОЖИДАЕТ ПРИНЯТИЯ

### 2. Специализации по типам сообщений
Агент адаптирует поведение в зависимости от контекста:

- **reminder**: Персонализированные напоминания о задачах
- **proactive**: Проактивные советы на основе предстоящих задач
- **result_check**: Проверка результатов выполнения задач
- **daily_report**: Ежедневные отчеты о прогрессе
- **overdue**: Обработка просроченных задач
- **system**: Системные уведомления (завершение, удаление задач)

### 3. Tool Calling и функции
Агент использует специализированные инструменты для выполнения действий:

- `create_task()`: Создание новой задачи
- `list_tasks()`: Получение списка задач с фильтрами
- `update_task()`: Обновление параметров задачи
- `delete_task()`: Удаление задачи
- `get_partners_list()`: Поиск контактов по интересам/навыкам
- `update_profile()`: Обновление профиля пользователя
- `update_user_memory()`: Сохранение долгосрочной памяти
- `delegate_task()`: Делегирование задач другим пользователям

### 4. Персонализация и профиль
- **Анализ профиля**: Учитывает интересы, навыки, цели пользователя
- **Адаптация под подписку**: Разные возможности для Bronze/Silver/Gold тарифов
- **Долгосрочная память**: Сохраняет паттерны поведения и предпочтения
- **Вовлечение**: Задает вопросы для заполнения профиля

### 5. Социальные функции
- **Поиск контактов**: Рекомендации партнеров по общим интересам
- **Делегирование**: Передача задач другим пользователям (Silver/Gold)
- **Сотрудничество**: Предложения для совместной работы

### 6. Интеллектуальный анализ
- **Паттерны поведения**: Анализирует повторяющиеся проблемы
- **Прогнозирование**: Предлагает решения на основе данных

## 7. Система логирования и аналитики

### Обзор системы логирования

AI-агент включает комплексную систему логирования для анализа работы, улучшения качества обслуживания и мониторинга производительности.

### Что логируется

#### AgentLog (таблица agent_logs)
Каждое взаимодействие с пользователем сохраняется в лог:

- **user_id**: ID пользователя
- **timestamp**: Время взаимодействия
- **message_type**: Тип сообщения (reminder, proactive, conversation, etc.)
- **user_message**: Исходное сообщение пользователя
- **agent_response**: Ответ агента
- **tools_used**: JSON массив использованных инструментов
- **task_actions**: JSON массив действий с задачами (created, completed, updated, deleted)
- **response_quality**: Автоматическая оценка качества ответа (1-10)
- **user_satisfaction**: Оценка пользователя (если предоставлена)
- **detected_intent**: Распознанное намерение пользователя
- **conversation_context**: Ключевые слова из контекста
- **response_time_ms**: Время генерации ответа
- **user_feedback**: Комментарии пользователя
- **improvement_suggestions**: Предложения по улучшению

#### AgentMetrics (таблица agent_metrics)
Агрегированные метрики по пользователям:

- **total_interactions**: Общее количество взаимодействий
- **tools_used_count**: Количество использованных инструментов
- **tasks_created/completed**: Статистика по задачам
- **avg_response_quality**: Средняя оценка качества ответов
- **avg_helpfulness**: Средняя полезность
- **common_intents**: Наиболее частые намерения
- **common_tools**: Наиболее используемые инструменты
- **error_patterns**: Паттерны ошибок

### Автоматическая оценка качества

Система автоматически оценивает качество ответов агента по шкале 1-10:

**Положительные факторы (+):**
- Длина ответа > 100 символов (+1)
- Живой стиль общения (+1)
- Наличие вопросов (+1)
- Уместные эмодзи (+0.5)

**Отрицательные факторы (-):**
- Слишком короткий ответ (< 20 символов) (-2)
- Переизбыток восклицаний (-0.5)

### Метрики производительности

Система предоставляет детальную аналитику:

```python
metrics = agent_logger.get_performance_metrics(user_id, days=30)
```

### Анализ паттернов

Система анализирует поведение пользователей:

- **Пиковые часы активности**: Определение наиболее активного времени
- **Общие темы**: Анализ наиболее частых тем разговоров
- **Эффективность ответов**: Процент высокооцененных ответов
- **Возможности улучшения**: Идентификация областей для оптимизации

### Практическое использование

#### Получение логов пользователя:
```python
logs = agent_logger.get_user_logs(user_id=12345, days=7)
for log in logs:
    print(f"{log.timestamp}: {log.user_message} -> {log.agent_response}")
```

#### Анализ производительности:
```python
patterns = agent_logger.analyze_patterns(user_id=12345)
print(f"Пиковые часы: {patterns['peak_hours']}")
print(f"Общие темы: {patterns['common_themes']}")
```

#### Добавление обратной связи:
```python
agent_logger.add_user_feedback(
    log_id=123,
    feedback="Очень полезный ответ!",
    rating=9
)
```

### Преимущества логирования

1. **Улучшение качества**: Анализ паттернов помогает выявлять слабые места
2. **Персонализация**: Понимание предпочтений пользователей
3. **Мониторинг**: Отслеживание работоспособности системы
4. **Оптимизация**: Данные для улучшения промптов и алгоритмов
5. **Аналитика**: Статистика использования функций и удовлетворенности

### Техническая реализация

- **Хранение**: PostgreSQL/SQLite таблицы с индексами
- **Производительность**: Асинхронное логирование без блокировки основного потока
- **Безопасность**: Шифрование чувствительных данных
- **Масштабируемость**: Агрегация метрик для больших объемов данных
- **Персонализированные советы**: Учитывает контекст и профиль
- **Адаптивность**: Изменяет подход в зависимости от ситуации

### 7. Безопасность и этика
- **Строгие правила**: Не создает задачи без подтверждения времени
- **Конфиденциальность**: Защищает личные данные
- **Этическое поведение**: Избегает навязчивых рекомендаций
- **Мониторинг**: Отслеживание подозрительной активности

### 8. Коммуникационный стиль
- **Conversational**: Использует живой язык ("знаешь", "кстати")
- **Эмодзи**: Уместное использование, но не в начале сообщений
- **Краткость**: 2-3 абзаца для полноценного ответа
- **Вовлечение**: Заканчивает вопросами для продолжения диалога

### 9. Интеграция с системой
- **Напоминания**: Автоматическая отправка уведомлений
- **Платежи**: Интеграция с Yookassa для подписок
- **База данных**: PostgreSQL для хранения данных
- **Redis**: Кеширование для производительности

### 10. Продвинутые возможности
- **Распознавание речи**: Транскрибация аудиосообщений
- **Многоязычность**: Поддержка русского языка
- **Временные зоны**: Корректная обработка времени
- **Статистика**: Отслеживание прогресса и метрик

## Архитектура агента

### Промпт-система
- **Базовый промпт**: Общие инструкции и правила
- **Специализации**: Динамическое добавление инструкций по типу сообщения
- **Контекст**: Передача профиля, задач, времени, подписки

### Обработка сообщений
1. Анализ входящего сообщения
2. Определение типа (reminder/proactive/etc.)
3. Формирование промпта с специализацией
4. Вызов AI с tool calling
5. Обработка ответа и выполнение действий
6. Отправка ответа пользователю

### Качество и тестирование
- **Unit-тесты**: Проверка всех функций
- **Интеграционные тесты**: Тестирование AI-ответов
- **Мониторинг**: Логирование и анализ ошибок
- **Валидация**: Проверка корректности tool calling

## Использование

Агент активируется автоматически при взаимодействии с ботом. Для оптимальной работы:
- Заполните профиль (интересы, навыки, цели)
- Указывайте точное время при создании задач
- Используйте естественный язык для общения
- Подтверждайте выполнение задач
