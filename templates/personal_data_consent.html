Соглашение об обработке персональных данных

Редакция от 1 января 2026 г.

1. Общие положения

Настоящее Соглашение определяет порядок обработки персональных данных пользователей сервиса ASI Biont (далее — «Сервис», «Оператор») в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

2. Оператор персональных данных

Оператором является физическое лицо — разработчик Сервиса. Контакт для обращений по вопросам персональных данных: @aleksandrinsider.

3. Какие данные мы обрабатываем

  • Идентификатор Telegram (user_id) и имя пользователя (username) — при регистрации через Telegram-бот
  • Идентификатор Discord (user_id) и имя пользователя — при регистрации через Discord-бот
  • Содержание задач, напоминаний и сообщений, которые вы передаёте боту
  • Часовой пояс и предпочтения уведомлений
  • История платежей (без данных карты — платёжные данные обрабатываются YooKassa и NowPayments)
  • IP-адрес при обращении к веб-интерфейсу для определения региона

4. Цели обработки данных

  • Предоставление функций Сервиса: управление задачами, напоминания, делегирование
  • Обработка платежей и начисление токенов
  • Отправка уведомлений в Telegram / Discord
  • Техническое обеспечение безопасности и работоспособности Сервиса
  • Определение режима отображения (язык, валюта оплаты) по IP-адресу

5. Правовые основания обработки

Обработка осуществляется на основании:

  • Согласия субъекта персональных данных (начало использования Сервиса означает принятие настоящего Соглашения)
  • Исполнения договора, стороной которого является пользователь (оказание услуг)

6. Хранение и защита данных

Данные хранятся на защищённых серверах Railway (США). Содержание задач шифруется. Доступ к данным имеет только Оператор. Данные не передаются третьим лицам, за исключением платёжных систем (только необходимый минимум для проведения платежа).

7. Срок хранения данных

Данные хранятся в течение всего срока использования Сервиса. При удалении аккаунта данные удаляются в течение 30 дней. Для удаления аккаунта обратитесь к Оператору: @aleksandrinsider.

8. Права пользователя

В соответствии с ФЗ-152 вы вправе:

  • Получить информацию об обрабатываемых персональных данных
  • Потребовать исправления неточных данных
  • Отозвать согласие на обработку с последующим удалением данных
  • Обратиться с жалобой в Роскомнадзор

Для реализации прав направьте обращение: @aleksandrinsider.

9. Файлы cookie и аналитика

Веб-интерфейс использует Яндекс.Метрику для анализа посещаемости. Метрика может устанавливать файлы cookie. Вы можете отключить сбор данных, используя настройки браузера или расширение Яндекс Браузера.

10. Изменения Соглашения

Оператор вправе вносить изменения в настоящее Соглашение. Актуальная версия всегда размещена по адресу /privacy. Продолжение использования Сервиса после публикации изменений означает их принятие.
