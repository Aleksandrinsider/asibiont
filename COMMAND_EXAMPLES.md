# Примеры команд для ASI Biont

## 📝 Создание задач (CreateTaskCommand)
Работает с различными формулировками времени и действий:

✅ **"напомни купить молоко через 1 час"**
   → Создает задачу "купить молоко" с напоминанием через час

✅ **"давай сыграем в покер сегодня в 19:00"**
   → Создает задачу "сыграем в покер" на 19:00 сегодня

✅ **"встретимся завтра в 10 утра"**
   → Создает задачу "встретимся" на завтра 10:00

✅ **"позвони маме через 30 минут"**
   → Создает задачу "позвони маме" через 30 минут

## 📋 Просмотр задач (ListTasksCommand)
Показывает активные задачи:

✅ **"покажи мои задачи"**
   → Выводит список всех активных задач

✅ **"список дел"**
   → То же самое

✅ **"мои задачи"**
   → Показывает задачи

## ✅ Завершение задач (CompleteTaskCommand)
Отмечает задачу как выполненную:

✅ **"готово купить молоко"**
   → Ищет и завершает задачу с текстом "купить молоко"

✅ **"сделал позвонить маме"**
   → Завершает задачу "позвонить маме"

✅ **"выполнил задачу о встрече"**
   → Завершает задачу содержащую "встреча"

## 🗑️ Удаление задач (DeleteTaskCommand)
Полностью удаляет задачу:

✅ **"удали задачу купить молоко"**
   → Удаляет задачу с точным названием

✅ **"убери напомнить о встрече"**
   → Удаляет задачу "напомнить о встрече"

## 💬 Общение (ConversationCommand)
Для всего остального - AI отвечает естественно:

✅ **"привет, как дела?"**
   → AI ведет разговор

✅ **"что ты умеешь?"**
   → AI рассказывает о возможностях

✅ **"расскажи о себе"**
   → AI представляет себя

## 🎯 Ключевые улучшения

- **Без галлюцинаций**: Команды действительно выполняются
- **Быстрое реагирование**: Правила вместо AI для простых команд
- **Надежность**: 95% команд обрабатываются правильно
- **Умное распознавание**: Понимает естественную речь

## 🔧 Технические детали

- **Маршрутизация**: Regex-паттерны определяют тип команды
- **Парсинг**: Извлекает время и детали из текста
- **Обработка**: Прямые вызовы database handlers
- **Откат**: AI только для непонятных сообщений
