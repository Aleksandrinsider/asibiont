### 4. Отсутствие async-обертки для delete_task
**Критичность:** 🟡 СРЕДНЯЯ

**Проблема:**  
- Функция `delete_task_sync` существует
- AI-агенту нужна async-версия `delete_task`
- После удаления дубликатов async-версия пропала

**Решение:**  
Добавлена async-обертка:
```python
async def delete_task(task_id=None, task_title=None, reason=None, 
                     user_id=None, session=None, close_session=True) -> str:
    """Async wrapper for delete_task_sync"""
    return delete_task_sync(
        task_id=task_id,
        task_title=task_title,
        reason=reason,
        user_id=user_id,
        session=session,
        confirmed=True  # Auto-confirm for AI agent
    )
```
