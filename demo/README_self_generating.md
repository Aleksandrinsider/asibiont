# Самообучающийся AI Агент

Этот проект демонстрирует революционный подход к созданию AI-агентов: **агент сам генерирует tool calls и код для их выполнения**.

## Концепция

Вместо предопределенных инструментов, агент:

1. **Анализирует запрос** через AI
2. **Сам определяет** необходимые действия
3. **Генерирует код** функций на лету
4. **Выполняет** сгенерированные функции
5. **Обучается** на каждом взаимодействии

## Архитектура

```
Запрос пользователя
        ↓
    AI Анализ + Планирование
        ↓
  Генерация кода функций
        ↓
  Динамическое выполнение
        ↓
    Адаптивный ответ
```

## Преимущества

- **Полная автономность**: Агент может решать любые задачи
- **Самообучение**: Каждый запрос улучшает способности агента
- **Гибкость**: Не ограничен предопределенными функциями
- **Адаптивность**: Может модифицировать свое поведение
- **Креативность**: Может "изобретать" новые подходы к решению задач

## Как это работает

### 1. Анализ запроса
AI получает запрос и создает план действий в формате JSON:
```json
{
    "actions": [
        {
            "type": "function_call",
            "name": "имя_функции",
            "code": "def func(...): ...",
            "parameters": {...}
        }
    ],
    "response_template": "Ответ с {placeholders}"
}
```

### 2. Генерация кода
AI генерирует Python код функций для выполнения конкретных задач.

### 3. Динамическое выполнение
Код выполняется в изолированном пространстве имен с доступом к проекту.

### 4. Кеширование функций
Сгенерированные функции сохраняются для повторного использования.

## Примеры возможностей

Агент может:
- ✅ Создавать любые функции на лету
- ✅ Выполнять SQL запросы к базе данных
- ✅ Делать API вызовы
- ✅ Анализировать данные
- ✅ Генерировать отчеты
- ✅ Решать нестандартные задачи

## Сравнение подходов

| Подход | Гибкость | Код | Автономность |
|--------|----------|-----|-------------|
| Жесткие команды | Низкая | 4000+ строк | Нет |
| Предопределенные tools | Средняя | 100 строк | Частичная |
| Самогенерация | Максимальная | 10 строк | Полная |
