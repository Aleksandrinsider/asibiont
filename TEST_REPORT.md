## 🔍 Доступные возможности агента

### 1. Работа с профилем (update_profile)
```python
# Добавление к спискам (навыки, интересы, цели)
update_profile(skills="Python, JavaScript")  # Добавит к существующим
update_profile(interests="спорт, музыка")    # Добавит к существующим

# Изменение полей (город, компания, должность, био, языки)
update_profile(city="Москва")
update_profile(company="Google", position="Senior Developer")

# Удаление из списков
update_profile(interests="-криптовалюты")    # Удалит из списка

# Полная очистка поля
update_profile(skills="")                     # Очистит все навыки
```

### 2. Работа с задачами
```python
# Создание задачи
add_task(title="Позвонить клиенту", reminder_time="2026-01-17 10:00")

# Завершение задачи
complete_task(task_title="Позвонить клиенту")  # По названию
complete_task(task_id=123)                     # По ID

# Удаление задачи
delete_task(task_title="Старая задача")        # По названию
delete_task(task_id=456)                       # По ID

# Удаление всех задач (с подтверждением пользователя)
delete_all_tasks()
```

### 3. Делегирование задач (delegate_task)
```python
# Доступно только для Silver и Gold тарифов
delegate_task(
    title="Подготовить отчет",
    delegated_to_username="sportfan2",
    reminder_time="завтра в 15:00"
)
```

### 4. Сохранение в память (update_user_memory)
```python
# Сохранение предпочтений, привычек и другой информации
update_user_memory(info="Предпочитает работать утром с 8 до 12")
```

### 5. Контекст диалога (Redis)
- Автоматически сохраняется при каждом сообщении
- Хранит последние 50 сообщений
- TTL: 24 часа
- Ключ: `context:{user_id}`

## 💡 Интеграция с UI

### Уведомления (showNotification)
Все действия пользователя через интерфейс показывают уведомления:

- **Делегирование** → "Запрос на делегирование отправлен агенту!" (success)
- **Ограничение тарифа** → "🥉 Делегирование доступно только на тарифах Серебро и Золото" (warning) + кнопка "Повысить тариф"
- **Выполнение задачи** → "Задача отмечена как выполненная" (success)
- **Удаление задачи** → "Задача удалена" (success)
- **Скрытие контакта** → "Контакт @username скрыт на 7 дней" (success)

### Стили тарифов
Унифицированы CSS классы для значков тарифов:
- `.tier-badge` - базовый стиль
- `.tier-badge-bronze` - желтый (#FEF3C7)
- `.tier-badge-silver` - серый (#E5E7EB)
- `.tier-badge-gold` - желтый (#FEF3C7)

Используется везде: в профиле, модальных окнах, списках контактов.

## 📦 Структура данных

### База данных (PostgreSQL/SQLite):
1. **users** - пользователи
2. **user_profiles** - профили пользователей
3. **tasks** - задачи
4. **interactions** - история сообщений
5. **subscriptions** - подписки
6. **promo_codes** - промокоды

### Redis:
- `context:{user_id}` - контекст диалога (50 сообщений, TTL 24ч)
- `message:{user_id}:{hash}` - дедупликация сообщений (TTL 30с)
