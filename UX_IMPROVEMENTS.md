# UX Анализ: Глазами Обычного Пользователя

## 🎯 Основные проблемы пользовательского опыта

### ❌ КРИТИЧЕСКИЕ ПРОБЛЕМЫ

#### 1. **Редактирование задач - слишком сложно**

**Проблема:**
```
Я: "Переименуй задачу про Петрова в срочный звонок"
Бот: "Для переименования мне нужно знать точное текущее название"
```

**Почему это плохо:**
- Пользователь **УЖЕ** указал "задачу про Петрова" - бот должен найти её сам
- Зачем спрашивать точное название, если я его и так не помню?
- Это как если бы вы сказали другу "переименуй ту задачу про звонок", а он ответил "скажи мне точное название"

**Ожидание:**
```
Я: "Переименуй задачу про Петрова в срочный звонок"
Бот: "✅ Готово! Переименовал 'Позвонить Петрову' → 'Срочный звонок'"
```

---

#### 2. **Удаление по ключевому слову не работает**

**Проблема:**
```
Я: "Удали задачу звонок"
Бот: "Не нашёл задачу 'звонок'. Может быть 'Позвонить...' или 'Звонок...'?"
```

**Почему это плохо:**
- У меня задача "Позвонить Петрову", я говорю "удали звонок" - это **очевидно** та же задача
- Поиск должен быть нечетким (fuzzy), а не точным
- Я не робот, я не помню точное название

**Ожидание:**
```
Я: "Удали задачу звонок"
Бот: "🗑️ Удалил задачу 'Позвонить Петрову'"
```

---

#### 3. **Завершение задачи - та же проблема**

**Проблема:**
```
Я: "Отметь задачу звонок выполненной"
Бот: "Давай уточним - какую задачу 'звонок' ты завершил?"
```

**Почему это плохо:**
- Если у меня только одна задача со словом "звонок" - зачем спрашивать?
- Даже если 2-3 задачи - покажи список и дай выбрать, а не заставляй вводить точное название

**Ожидание:**
```
Вариант 1 (одна задача):
Я: "Отметь звонок выполненным"
Бот: "✅ Отлично! Задача 'Позвонить Петрову' завершена"

Вариант 2 (несколько задач):
Я: "Отметь звонок выполненным"
Бот: "Нашёл 2 задачи со словом 'звонок':
1. Позвонить Петрову (завтра 15:00)
2. Звонок в банк (сегодня 16:00)
Какую отметить выполненной?"
```

## 📝 КОНКРЕТНЫЕ FIX'Ы

### Fix 1: Улучшить find_task_by_title()

**Текущий код:**
```python
task = session.query(Task).filter(
    Task.user_id == user.id,
    Task.title.ilike(f"%{task_title}%")
).first()
```

**Проблема**: Ищет точное вхождение подстроки

**Решение:**
```python
def find_task_fuzzy(task_title, user, session):
    """Нечеткий поиск задачи"""
    # 1. Точное совпадение
    exact = session.query(Task).filter(
        Task.user_id == user.id,
        Task.title.ilike(task_title)
    ).first()
    if exact:
        return exact
    
    # 2. Содержит подстроку
    contains = session.query(Task).filter(
        Task.user_id == user.id,
        Task.title.ilike(f"%{task_title}%")
    ).all()
    if len(contains) == 1:
        return contains[0]
    elif len(contains) > 1:
        return contains  # Вернуть список для выбора
    
    # 3. Fuzzy match по словам
    words = task_title.lower().split()
    for task in session.query(Task).filter_by(user_id=user.id).all():
        task_words = task.title.lower().split()
        matches = sum(1 for w in words if any(w in tw for tw in task_words))
        if matches >= len(words) * 0.6:  # 60% совпадение
            return task
    
    return None
```
