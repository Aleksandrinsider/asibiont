# План улучшения общения агентов в чате

## Текущее состояние (после анализа кода)

### Поток сообщений агентов
```
Пользователь → chat_with_ai() [chat.py] → autonomous_agent.chat_with_ai() [autonomous_agent.py]
  ├── _office_director_chat() — решает: делегировать или ответить самому
  │   ├── _run_agent_task() → _exec_agent_for_director() → агент выполняет
  │   └── Возвращает результат → ASI подводит итог
  └── process_request() — если директор не перехватил
```

### Проблемы (6 ключевых)

1. **Директивы обрезаются до 130 символов** (строка 12597-12598) — пользователь видит `"Кристина, проверь почту..."` вместо полного поручения

2. **Отчёты агентов проходят через regex-мясорубку** (`_live_result_text`) — из результата оставляют 1-2 предложения, убирают стиль, эмодзи, структуру

3. **Нет объяснения "почему этот агент"** — ASI просто пишет поручение, не объясняя пользователю свой выбор

4. **Агенты не показывают свой характер** — personality агента используется только в `_exec_agent_for_director`, но результат проходит через общую очистку

5. **Нет промежуточных статусов** — агент не говорит "начал работу", "нашёл первые данные", "почти готов"

6. **Отсутствует диалог между агентами** — multi_agent.py — это только внутренние "режимы мышления", а не реальные персонажи

## План изменений

### Изменение 1: Естественные директивы (handlers.py — delegate_task)
**Файл:** `ai_integration/handlers.py`, функции `_live_assignment_text` и `delegate_task`

**Что сделать:**
- Убрать regex-нормализацию инфинитивов (`_inf_map`) — AI сам умеет писать в повелительном наклонении
- Упростить `_strip_structured_text` — не вырезать блоки, а сохранять структуру (заголовки, шаги)
- Убрать обрезание по `_max_len` — AI знает пропорции
- Сохранять полный текст директивы в Interaction, а не обрезанную версию

**Режим PURE_LIVE уже включён** (`_pure_live` в `sanitize_live_team_chat_text`) — просто убрать `_strip_structured_text` блокировку.

**Результат:** Директива выглядит как сообщение человека коллеге, а не как API-запрос.

### Изменение 2: Контекст выбора агента (autonomous_agent.py — _office_director_chat)
**Файл:** `ai_integration/autonomous_agent.py`

**Что сделать:**
- В функции `_run_agent_task` добавить перед директивой короткое объяснение: `"Кристина разбирается с email — ей лучше поручить"`
- Показывать пользователю не обрезанную директиву (130 символов), а 2-3 предложения с контекстом
- Использовать имя агента и его специализацию в объяснении

**Логика:** 
```python
# Вместо: "Кристина, проверь почту..."
# Показывать: "Кристина как раз отвечает за почту → попросил её проверить входящие. Она найдёт письма от партнёров."
```

### Изменение 3: Живые отчёты агентов (handlers.py — _live_result_text)
**Файл:** `ai_integration/handlers.py`, функция `_live_result_text`

**Что сделать:**
- Убрать обрезание "1-2 предложения" (`_sent[:2]`) — агент пишет столько, сколько нужно
- Убрать `_max_len=400` и `_truncate_by_word(280)` — AI сам контролирует длину
- Убрать принудительное удаление эмодзи и markdown — агент использует их если уместно
- Оставлять структуру: если агент написал "Нашла: Х, Y, Z" — не превращать это в "Нашла X Y Z"

**Результат:** Агент пишет своим голосом, с деталями, как человек отчитывается перед руководителем.

### Изменение 6: Системный промпт — новые правила (system_prompt.py)
**Файл:** `ai_integration/system_prompt.py`

**Что сделать (добавить в секцию "Агенты"):**

Добавить правила:
- "Когда даёшь поручение агенту — объясни пользователю почему выбрал этого агента: 'Хьюго специалист по email, передал ему'"
- "Когда агент вернул результат — перескажи своими словами: 'Кристина нашла 3 контакта: один по X, два по Y. Что думаешь?'"  
- "НЕ пиши технически: 'делегировал задачу агенту', 'запустил инструмент'. Пиши: 'попросил Кристину проверить почту', 'Хьюго ищет варианты'"
- "Агенты — это твои коллеги, не инструменты. Говори о них как о членах команды: 'Кристина подтвердила', 'Хьюго нашёл интересное'"

## Итоговая структура сообщения в чате (как должно выглядеть)

### Сейчас:
```
ASI: Кристина, проверь почту...
Кристина: Вот что я нашел: пока данных мало, продолжаю проверку.
ASI: Кристина проверила. Ничего нового.
```

### После изменений:
```
ASI: Хьюго сейчас как раз работает с email — попросил его проверить входящие и найти новых партнёров. У него подключена почта, быстро справится.
Хьюго: Заглянул в почту. Есть 3 новых письма:
- Ответ от Ивана из «ТехноСтарт» — интересуется партнёркой
- Счёт от сервиса аналитики — оплачен до конца месяца
- Спам-рассылка — пропустил
Что по Ивану — напишу завтра утром? Самый перспективный контакт.
ASI: Хьюго разобрал почту. Есть тёплый контакт — Иван из ТехноСтарта, заинтересован в партнёрской программе. Написать ему завтра?
```
