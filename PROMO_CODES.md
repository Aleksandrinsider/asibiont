# Промокоды

## Созданные промокоды

Система промокодов позволяет активировать подписку на 30 дней бесплатно. Каждый пользователь может использовать промокод только один раз.

### Доступные промокоды

| Код | Тариф | Длительность | Скидка | Срок действия | Лимит |
|-----|-------|--------------|--------|---------------|-------|
| **LIGHT1** | 🥉 Бронза (LIGHT) | 30 дней | 100% | до 31.12.2026 | ∞ |
| **STD2026XPRO** | 🥈 Серебро (STANDARD) | 30 дней | 100% | до 31.12.2026 | ∞ |
| **PREM2026ELITE** | 🥇 Золото (PREMIUM) | 30 дней | 100% | до 31.12.2026 | ∞ |
| **VIPACCESS2026** | 🥇 Золото (PREMIUM) | **365 дней (1 год)** | 100% | до 31.12.2026 | **1 использование** |

## Как использовать

### В Telegram боте

1. Отправьте команду `/promo <КОД>`
2. Например: `/promo LIGHT1`
3. Подписка активируется автоматически

### Команды

- `/promo` - показать список доступных промокодов
- `/promo LIGHT1` - активировать промокод LIGHT1
- `/subscribe` - открыть страницу подписки
- `/dashboard` - посмотреть статус подписки

## Ограничения

- ✅ Один пользователь может использовать промокод только один раз
- ✅ Промокод привязан к конкретному тарифу
- ✅ После активации подписка действует 30 дней (или 365 дней для VIPACCESS2026)
- ✅ Промокоды действуют до 31.12.2026
- ⚠️ **VIPACCESS2026** - специальный одноразовый код, может быть использован только **одним** пользователем!

## Администрирование

### Создание новых промокодов

Используйте скрипт `create_promo_codes.py`:

```python
python create_promo_codes.py
```

### Проверка использования

Промокоды хранятся в таблице `promo_codes`:
- `code` - код промокода
- `used_count` - количество использований
- `used_by_users` - JSON-массив ID пользователей, использовавших код
- `max_uses` - максимальное количество использований (NULL = неограничено)

### История активаций

Все активации записываются в таблицу `payment_history`:
- `action` = 'promo_activated'
- `details` содержит информацию о промокоде

## Технические детали

### Структура базы данных

```sql
CREATE TABLE promo_codes (
    id SERIAL PRIMARY KEY,
    code VARCHAR(50) UNIQUE NOT NULL,
    tier VARCHAR(20) NOT NULL,
    discount_percent INTEGER DEFAULT 0,
    max_uses INTEGER,
    duration_days INTEGER DEFAULT 30,
    expires_at TIMESTAMP NOT NULL,
    used_count INTEGER DEFAULT 0,
    used_by_users TEXT DEFAULT '[]',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Логика активации

1. Проверка существования промокода
2. Проверка, не использовал ли пользователь этот промокод ранее
3. Проверка лимита использований
4. Проверка срока действия
5. Активация или обновление подписки
6. Обновление счетчика использований
7. Запись в историю платежей

## Примеры использования

### Пользователь активирует промокод

```
Пользователь: /promo LIGHT1

Бот: ✅ Промокод активирован!

🎁 Тариф: 🥉 Бронза
📅 Действует до: 30.02.2026 15:30

Используйте /dashboard для просмотра подписки
```

### Повторная попытка использовать промокод

```
Пользователь: /promo LIGHT1

Бот: ❌ Вы уже использовали этот промокод.
```

### Неверный промокод

```
Пользователь: /promo INVALID

Бот: ❌ Промокод не найден. Проверьте правильность ввода.
```
