## Примеры работы

### Парсинг времени:
```python
"завтра в 10:00" → 2026-01-29 10:00 MSK
"послезавтра в 14:30" → 2026-01-30 14:30 MSK
"через 2 часа" → now + 2 hours
"15:30" → сегодня/завтра в 15:30
```

### Поиск задач:
```python
Задача: "Проверить почту"
Найдет: "проверь почту", "почта готова", "закрой почту"

Задача: "Встреча с командой"
Найдет: "встреча", "встречу", "командой"
```

## Технические детали

### Stemming
Простое удаление типичных русских окончаний:
- а, я, у, ю, ы, и
- "почта" → "почт", "почту" → "почт" → совпадение!

### AI-парсинг
```json
Промпт: "Текущее время: 2026-01-28 02:34 (вторник)
Пользователь хочет: 'завтра в 10:00'
Верни JSON с целевым временем"

Ответ: {"year": 2026, "month": 1, "day": 29, "hour": 10, "minute": 0}
```

## Преимущества

1. **Нет жестких шаблонов** - AI понимает контекст
2. **Русские окончания** - stemming решает проблему падежей
3. **Универсальность** - одна система для всех команд
4. **Надежность** - fallback на простой парсинг
5. **Логирование** - детальные логи для отладки
