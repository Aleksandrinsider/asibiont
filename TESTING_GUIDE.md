# 🧪 Инструкция по тестированию

## Быстрый запуск тестов

### 1. Тест улучшений AI (рекомендуется)
```bash
python test_ai_improvements.py
```
**Проверяет:**
- ✅ Точное время вместо расплывчатого
- ✅ Локальное время (не UTC)
- ✅ Отсутствие избыточных упоминаний контактов

### 2. Интерактивный веб-чат
```bash
# Сначала запустите сервер в отдельном терминале:
python main.py

# Затем в другом терминале:
python test_web_chat.py
```

**Режимы работы:**
```bash
# Интерактивный чат
python test_web_chat.py

# Автоматический сценарий
python test_web_chat.py auto

# Отправить одно сообщение
python test_web_chat.py message "привет, как дела?"
```

### 3. Комплексный продакшн-тест (долго)
```bash
python test_production_ready.py
```
**Проверяет:**
- Базовые задачи
- Делегирование
- Профиль и нетворкинг
- Время и напоминания
- Удаление с причиной
- Проактивность
- Память и контекст
- Граничные случаи
- Подтверждения

## Требования перед тестированием

1. **Убедитесь что установлены зависимости:**
```bash
pip install -r requirements.txt
```

2. **Настройте переменные окружения:**
```bash
LOCAL=1  # Для локального тестирования
FREE_ACCESS_MODE=1  # Опционально, для бесплатного доступа
```

3. **База данных:**
   - Для локального тестирования используется SQLite (`test_db.sqlite`)
   - Автоматически создается при первом запуске `python main.py`

## Что проверять вручную

### Критические функции:
1. **Создание задачи с временем:**
   - "Напомни через 10 минут позвонить Сергею"
   - Должно показать точное время (например "20:05")

2. **Текущее время:**
   - "Сколько сейчас времени?"
   - Должно показать локальное время без UTC

3. **Делегирование:**
   - "Делегируй @username подготовить отчет"
   - Должен спросить про результат и критерии

4. **Удаление:**
   - "Удали задачу X, она больше не нужна"
   - Должен учесть причину в ответе

5. **Завершение:**
   - "Позвонил Сергею, обсудили проект"
   - Должен отметить выполнение и учесть детали

## Ожидаемые результаты

### ✅ Успешный тест:
- AI указывает точное время
- Нет упоминаний UTC
- Не избыточные упоминания контактов
- Учитывает причины и результаты

### ❌ Неудачный тест:
- "к вечеру" вместо "20:00"
- "16:45 по UTC" вместо локального времени
- Многократные упоминания контактов
- Не учитывает причины удаления

## Troubleshooting

### Ошибка: "REMINDER_SERVICE not initialized"
**Решение:** Это нормально для тестов без запущенного reminder_service. Задачи создаются, но напоминания не отправляются.

### Ошибка: "Session not bound"
**Решение:** Перезапустите тест. Проблема с SQLAlchemy сессиями.

### Ошибка: "Для использования требуется активная подписка"
**Решение:** Установите `FREE_ACCESS_MODE=1` в переменных окружения или активируйте подписку через админ-панель.
