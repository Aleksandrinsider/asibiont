## Выявленные проблемы

### 1. Не отображаются делегированные задачи в "Поручил я"
**Симптомы:** 
- Контакт @fitness_maria виден в разделе "Поручаю я"
- Но задачи делегированные на неё не отображаются в списке задач

**Возможные причины:**
- Фильтрация задач по статусу
- Неправильная логика отображения делегированных задач
- Проблема в frontend рендеринге

### 2. Выполнение одной задачи закрывает другие
**Симптомы:**
- Пользователь выполнил задачу делегированную от @sport_alex
- Одновременно закрылась личная задача пользователя

**Возможные причины:**
- Неправильная идентификация задачи (ID конфликт)
- Логика обновления статуса применяется ко всем задачам
- Проблема в SQL запросе (WHERE условие неправильное)
