## ✅ Что сделано

### 1. Созданы 3 промокода
- **LIGHT1** - месяц подписки LIGHT (🥉 Бронза)
- **STD2026XPRO** - месяц подписки STANDARD (🥈 Серебро)

### 2. Характеристики промокодов
- ✅ Скидка 100% (бесплатная активация)

## 🚀 Использование

### Для пользователей

1. Откройте бота в Telegram
2. Введите `/promo LIGHT1` (или другой код)
3. Подписка активируется автоматически на 30 дней
4. Проверьте статус: `/dashboard`

### Примеры диалогов

**Успешная активация:**
```
Пользователь: /promo LIGHT1

Бот: ✅ Промокод активирован!

🎁 Тариф: 🥉 Бронза
📅 Действует до: 30.02.2026 15:30

Используйте /dashboard для просмотра подписки
```

**Повторное использование:**
```
Пользователь: /promo LIGHT1

Бот: ❌ Вы уже использовали этот промокод.
```

**Неверный код:**
```
Пользователь: /promo INVALID

Бот: ❌ Промокод не найден. Проверьте правильность ввода.
```

**Помощь:**
```

Например: /promo LIGHT1

Доступные промокоды:
• LIGHT1 - месяц подписки LIGHT

• PREM2026ELITE - месяц подписки PREMIUM
```

## 📊 Мониторинг

### Проверка статистики

```python
python create_promo_codes.py
```

Выведет список всех промокодов с количеством использований.

### SQL запросы

```sql
-- Все промокоды
SELECT * FROM promo_codes;

-- Статистика использований
SELECT code, tier, used_count, 
       json_array_length(used_by_users::json) as unique_users
FROM promo_codes;

-- История активаций
SELECT * FROM payment_history 
WHERE action = 'promo_activated' 
ORDER BY created_at DESC;
```
