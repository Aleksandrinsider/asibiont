# Автоматические посты - Руководство

## Что добавлено:

### 1. **Поле даты рождения** 📅
- Добавлено обязательное поле "Дата рождения" в форму регистрации
- Формат: ДД.ММ.ГГГГ (например: 15.03.1990)
- Автоматический расчет знака зодиака при вводе
- Отображается в профиле пользователя

### 2. **Автоматические посты о прогрессе** 📊
- Сервис создает случайные посты от имени пользователя о выполнении задач
- Посты генерируются AI на основе реальной статистики:
  - Сколько задач создано
  - Сколько выполнено
  - Сколько просрочено
- Посты мотивирующие и вовлекающие в диалог
- Создаются в случайное время между 12:00 и 22:00 (шанс 20% при каждой проверке)
- Не более 1 поста в день от одного пользователя

### 3. **Поздравительные посты в день рождения** 🎂
- Автоматически создается пост в день рождения пользователя
- Пост генерируется AI с учетом возраста и знака зодиака
- Приглашает других пользователей присоединиться к празднованию

## Как запустить сервис автопостов:

### Локально (для тестирования):
```bash
# Windows PowerShell
$env:LOCAL="1"; python auto_post_service.py

# Linux/Mac
LOCAL=1 python auto_post_service.py
```

### На продакшене (Railway):
Добавьте в Procfile:
```
web: python main.py
worker: python auto_post_service.py
```

Или запустите как фоновый процесс:
```bash
nohup python auto_post_service.py > auto_post.log 2>&1 &
```

## Настройки сервиса:

В `auto_post_service.py` можно настроить:
- **Интервал проверки**: `await asyncio.sleep(3600)` - по умолчанию каждый час
- **Шанс создания поста**: `if random.random() < 0.2:` - по умолчанию 20%
- **Время постов**: `if 12 <= current_hour <= 22:` - с 12:00 до 22:00

**Прогресс:**
- "Сегодня закрыл 5 задач из 7, неплохо! 💪 Осталось доделать пару мелочей. Как у вас день прошёл?"

- "🎂 День рождения! Официально стал на год опытнее. Принимаю поздравления и советы!"

## База данных:

Миграция выполнена автоматически, добавлены поля:
- `birthdate` VARCHAR(10) - дата рождения в формате ДД.ММ.ГГГГ
- `zodiac_sign` VARCHAR(20) - знак зодиака (рассчитывается автоматически)

## Проверка работы:

1. Откройте дашборд, если профиль не заполнен - увидите форму с полем даты рождения
2. Заполните дату рождения, знак зодиака отобразится автоматически
3. Запустите сервис автопостов
4. Проверьте логи - сервис будет проверять пользователей каждый час
5. В ленте новостей появятся автоматические посты

## Устранение неполадок:

**Проблема**: Сервис не создает посты
- Проверьте что у пользователей заполнен профиль (город обязателен)
- Проверьте что есть задачи в базе
- Проверьте логи сервиса на ошибки

**Проблема**: Дата рождения не сохраняется
- Проверьте формат ввода: ДД.ММ.ГГГГ
- Проверьте что миграция выполнена (поля добавлены в БД)
- Проверьте логи backend

**Проблема**: Знак зодиака не рассчитывается
- Проверьте правильность даты (день, месяц, год в допустимых пределах)
- Проверьте JavaScript console на ошибки
